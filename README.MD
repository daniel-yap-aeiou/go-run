How to run:
<br/>
 1. Open VS Code
 2. Open Terminal
 3. Enter 'cd go-mysql-app'
 3. Enter 'go run main.go' into the terminal console (Make sure your firewall has allow this code to run)
 4. Open web browser, enter 'localhost:8090'
